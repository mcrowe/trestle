# Trestle: Git-flavoured templates for Rails apps

## Setup
- `./bin/setup`
- `bundle`
- `rake db:create:all`
- Set `GOOGLE_ANALYTICS_SITE_ID` environment variable
- Set the `SECRET_KEY_BASE` environment variable to a new secret (`rake secret`).

## Usage

Run tests using `rake test:all`
